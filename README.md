# Opofinance Marketing Dashboard

A frontend-only marketing dashboard concept for a forex brokerage team.

This project was built as an AI-assisted homework exercise using pure HTML, CSS, and JavaScript. It includes multiple pages, a modern glassmorphism-inspired UI, and a responsive dashboard layout focused on marketing performance.

## Project Pages

- `index.html` — Login page
- `signup.html` — Sign up page
- `dashboard.html` — Main dashboard page

## Main Features

- Frontend-only implementation with no backend and no database
- Separate pages for login, signup, and dashboard
- Simulated authentication flow using JavaScript
- Responsive layout for desktop and smaller screens
- Marketing-focused dashboard widgets and sections
- Dark and light mode toggle on the dashboard
- Modern sidebar and top navigation layout

## Dashboard Highlights

- Lead momentum overview
- Campaign ROI and conversion metrics
- Conversion goal progress card
- Weekly performance chart
- Top acquisition channels
- Recent campaign activity timeline

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts (`Space Grotesk`)

## How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Use any email and password to enter the dashboard

## Notes

- This project is a UI demo and does not connect to a backend service
- Form submission and login behavior are simulated for presentation purposes
- User name and theme preference are stored in the browser with local storage

## Goal of the Project

The goal of this project is to demonstrate how AI tools can help create a clean and creative frontend dashboard with structured pages, consistent UI, and clear visual storytelling for a marketing workflow.
